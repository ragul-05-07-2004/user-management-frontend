# User Management Frontend

This is the frontend of the User Management System built using **React (Vite)**.  
It allows users to **register, login, and manage their profile**.

---

## Live Demo

Frontend URL  
https://rahul-full-stack-project.vercel.app

Backend API  
https://user-management-api-mace.onrender.com

---

## Tech Stack

- React (Vite)
- Axios
- CSS
- React Router

---

## Features

- User Registration
- User Login
- Profile Page
- Update Profile
- Profile Image Upload
- Form Validation
- Toast Notifications
- Responsive UI
